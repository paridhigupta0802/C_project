# Cafeteria Billing System

A modular Cafeteria Billing System developed in C using file handling and structured programming concepts.

## Features
- User Authentication
- Role-based Access (Admin & Manager)
- Menu Management
- Order Processing
- Billing System
- Bill History & Record Management
- Daily Sales Calculation

## Technologies Used
- C Programming
- File Handling
- Structures
- Modular Programming

## Modules
- login.c
- menu.c
- order.c
- billing.c
- record.c
- main_driver.c

## Compile Command

```bash
gcc *.c -o cafe
